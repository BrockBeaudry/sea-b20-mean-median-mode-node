Mean, Median, and Mode
========================

A simple node application which calculates the mean, median, and mode of a series of numbers.

### Usage

Enter the numbers as command line arguments.
```
node mean-median-mode 1 2 3 4 1
```

Output:
```
Mean: 2.2
Median: 2
Mode: 1
```
